# UI Polish

Improve UX clarity without breaking existing interaction patterns.

## Steps
- Prioritize task-critical affordances (search, attach, restore, send, stop).
- Add visual hierarchy for high-signal states: running, errors, usage, context pressure.
- Keep mobile and narrow layouts readable.
- Ensure every new control has a fast keyboard path when possible.

## Output
- List of UI improvements with user benefit.
- Any accessibility or responsiveness notes.
- Screenshot-equivalent textual walkthrough of the updated flow.
