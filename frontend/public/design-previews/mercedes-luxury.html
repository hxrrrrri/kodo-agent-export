04 — Cards & Containers
Content Surfaces
Cards use surface token on canvas background. 12px radius, hairline border, and subtle shadow creates depth without ornamentation.
AUTOMOTIVE · FEATURE
Design with Clarity
A system built on restraint. Every token earns its place through function, not decoration.
AUTOMOTIVE · GUIDE
Token Architecture
Five core tokens map to semantic roles. Primitive → semantic → component. No raw hex values in components.
AUTOMOTIVE · MOTION
Interaction Patterns
150ms ease-out for micro-interactions. Hover states use accent-alpha overlay. Focus rings use 3px accent glow.
05 — Form Elements
Input Components
Inputs use 8px radius and surface background. Focus state thickens border to accent color with 3px glow ring.
